# SkyRoads-inspired research portfolio

A lightweight static website for GitHub Pages. It uses only HTML, CSS, and JavaScript, so there is no build step.

## Files

- `index.html` — content and page sections
- `styles.css` — visual design
- `script.js` — animated starfield, pseudo-3D road, and HUD meters
- `favicon.svg` — simple original icon
- `.nojekyll` — tells GitHub Pages to serve the files directly

## How to deploy on GitHub Pages

1. Create a repository named `YOUR-USERNAME.github.io`.
2. Copy these files into the root of the repository.
3. Commit and push to the `main` branch.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select `main` and `/ (root)`, then save.
7. Visit `https://YOUR-USERNAME.github.io`.

## Customize

Replace the placeholder name, email, GitHub, Google Scholar, project cards, and publication entries in `index.html`.

This design is an original homage to retro space-road games. It does not include original SkyRoads art, audio, level files, or logos.
